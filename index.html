About

I'm Srija

Numbers have always fascinated me, but it was in my B.Tech—when I first turned a mess of data into a simple chart—that I realized data could tell stories. That spark carried me into my M.S. in Data Analytics Engineering, where I dove headfirst into building end-to-end pipelines that wrangle raw data into clean, queryable tables; training machine-learning models that forecast trends; and crafting interactive dashboards that transform complexity into clarity. Whether I’m optimizing a SQL query, prototyping a Python script to detect anomalies, or bending React to my will to present insights in a beautiful UI, I thrive on turning abstract data into real-world impact.


But the magic doesn’t stop at code. I love teaming up with others—whether I’m coding side-by-side with designers to refine a flow, hashing out priorities with analysts, or breaking down data insights so everyone (even non-tech folks) can follow. I believe the best solutions emerge when everyone speaks the same language, and I love bridging those gaps. Now, I’m looking for a role where I can blend this academic rigor, technical agility, and human-centered mindset—building data-driven experiences that not only answer today’s questions, but spark tomorrow’s innovations.
